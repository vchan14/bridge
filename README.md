# bridge
A web application that bridges two people to learn each other language. 

## Features 
- [P] Landing Page : http://bit.ly/bridgeus
- [] Signup and login 
- [] Avability, language preference, and proficient options
- [] Common chatroom 
- [] Video Chat 
- [] Lesson prompts
